这是一个 [Next.js](https://nextjs.org/) 项目.

## 开始

启动开发服务器:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)

找到页面编辑查看实时结果 `app/page.tsx`.

## 构建

```bash
# 测试环境构建
npm run build-test
# 正式环境构建
npm run build-prod
# or
yarn build
# or
pnpm build
# or
bun build
```

## 启动

启动时指定的端口号是`8888`

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

更多关于`nextjs`的部署见[Next.js deployment documentation](https://nextjs.org/docs/deployment)
